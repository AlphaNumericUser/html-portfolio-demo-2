Archeoexplorer

Archeoexplorer es una aplicación móvil y web que permite a los usuarios explorar sitios arqueológicos. La aplicación muestra información sobre estos sitios, como su ubicación, características especiales y fotografías.

Esta aplicación fue desarrollada con Flutter y Firebase como base de datos.

Características más destacadas:

  • Extracción de coordenadas de las imágenes mediante una API propia hecha en Python
  • Uso de In-App Purchases para gestionar suscripciones y donaciones
  • Integración de anuncios con AdMob
  • Reconocimiento de imágenes con Google Cloud Vision
  • Moderación de contenido con IA mediante Perspective API
  • Chat en tiempo real para el centro de ayuda
  • Capacidad de mostrar la aplicación en más de 10 idiomas
